# ประกาศเรื่องความเป็นส่วนตัวของระบบปฏิบัติการ Firefox

26 สิงหาคม 2015
{: datetime="2015-08-26" }

Mozilla ใส่ใจกับความเป็นส่วนตัวของคุณ [นโยบายความเป็นส่วนตัวของ Mozilla](https://www.mozilla.org/privacy/) ของเราอธิบายถึงวิธีที่เราปฏิบัติต่อข้อมูล เมื่อระบบปฏิบัติการ Firefox ส่งข้อมูลนั้นมาที่เรา

## สิ่งที่คุณควรทราบ

ระบบปฏิบัติการ Firefox เชื่อมต่อกับเราและผู้อื่นก็เพื่อมอบคุณสมบัติต่างๆ ของผลิตภัณฑ์
{: #location-services }

* การอัพเดท: เพื่อให้ระบบอัพเดทอยู่เสมอ ระบบปฏิบัติการ Firefox จะเชื่อมต่อโดยอัตโนมัติกับบริการที่ดำเนินการโดยหรือในนามของผู้ผลิตอุปกรณ์ของคุณ นอกจากนี้ ระบบปฏิบัติการ Firefox จะตรวจสอบรายการอัพเดทของแอพต่างๆ ที่ได้ที่ติดตั้งไว้ โดยส่งคำขอไปที่ Firefox Marketplace หรือที่บริการของผู้พัฒนาแอพนั้นๆ หากคุณติดตั้งแอพจากตำแหน่งที่ตั้งอื่น
* ตำแหน่งที่ตั้ง: เมื่อคุณเปิดบริการหาตำแหน่งที่ตั้ง ระบบปฏิบัติการ Firefox อาจใช้จุดของข้อมูลและเอนทิตีจำนวนมากในการประมาณการตำแหน่งที่ตั้งของคุณ รวมถึง GPS, AGPS ตลอดจนผู้ให้บริการระบุพิกัดอื่นๆ คุณสามารถปิดใช้งานบริการหาตำแหน่งที่ตั้งได้ในแอพ “การตั้งค่า” ของระบบปฏิบัติการ Firefox ในอุปกรณ์ของคุณ [ดูข้อมูลเพิ่มเติมเกี่ยวกับข้อกำหนดของผู้ให้บริการ AGPS](https://wiki.mozilla.org/Firefox_OS/AGPS_service_provider_terms)
* การเปิดใช้งานครั้งแรก: เมื่อเปิดใช้งานอุปกรณ์ที่ใช้ระบบปฏิบัติการ Firefox เป็นครั้งแรก ระบบปฏิบัติการ Firefox จะส่งการปิงข้อมูลแบบครั้งเดียว (เช่น ข้อมูลเกี่ยวกับข้อมูลจำเพาะของอุปกรณ์ เช่น เวอร์ชั่นที่ออกใหม่ของซอฟต์แวร์, ID คู่ค้า, ประเทศ และขนาดของหน้าจอ) ไปยัง Mozilla ซึ่งจะช่วยให้ Mozilla พิจารณาได้ว่าจำเป็นต้องมีการอัพเดตความปลอดภัยหรือการซ่อมแซมที่จำเป็นสำหรับผู้ใช้และคู่ค้าของเราหรือไม่ พิจารณาถึงความต้องการต่างๆ เพื่อให้นักพัฒนาแอพสามารถสร้างแอพที่ดียิ่งขึ้นเพื่อผู้ใช้ของเรา และเอื้อต่อการวิเคราะห์ตลาดของ Mozilla (เช่น พิจารณาว่ามีอุปกรณ์ที่เปิดใช้งานอยู่จำนวนเท่าไร) [ดูข้อมูลเพิ่มเติม](https://wiki.mozilla.org/Firefox_OS/Metrics/activationping)
* การใช้งานแอพ: เพื่อให้พบเนื้อหาใน Firefox Marketplace ได้ดียิ่งขึ้น ระบบปฏิบัติการ Firefox จะส่ง "ข้อมูลการใช้งานแอพพลิเคชั่น" (เช่น จำนวนครั้งที่เปิดใช้งาน จำนวนการเปิด/ปิด/หยุดทำงาน รายชื่อแอพที่ถอนการติดตั้งบนอุปกรณ์ที่ใช้ระบบปฏิบัติการ Firefox) ไปยัง Mozilla คุณสามารถปิดใช้งานการแชร์ข้อมูลการใช้งานแอพได้ในแอพ “การตั้งค่า” ของระบบปฏิบัติการ Firefox ในอุปกรณ์ของคุณ [ดูข้อมูลเพิ่มเติม](https://wiki.mozilla.org/FirefoxOS/Metrics/App_Usage)
* ข้อมูลผู้ให้บริการเสิร์ชเอนจิน: เพื่อปรับปรุงประสิทธิภาพการค้นหาบนอุปกรณ์ที่ใช้ระบบปฏิบัติการ Firefox ระบบปฏิบัติการ Firefox จะส่งข้อมูลไปยัง Mozilla ว่าเสิร์จเอนจินใดที่ส่งผลการค้นหาไปที่อุปกรณ์ คุณสามารถปิดใช้งานการแชร์ข้อมูลผลการค้นหาของเสิร์ชเอนจินได้ในแอพ “การตั้งค่า” ของระบบปฏิบัติการ Firefox ในอุปกรณ์ของคุณ [ดูข้อมูลเพิ่มเติม](https://wiki.mozilla.org/FirefoxOS/Metrics/App_Usage)

---------------------------------------

อุปกรณ์ที่คุณกำลังใช้นั้นสร้างขึ้นโดยความร่วมมือจากหลายฝ่าย เช่นเดียวกับอุปกรณ์เคลื่อนที่ส่วนใหญ่ ประกาศฉบับนี้ครอบคลุมเฉพาะเนื้อหาเกี่ยวกับวิธีที่ Mozilla ปฏิบัติต่อข้อมูลของคุณสำหรับระบบปฏิบัติการ Firefox ในเวอร์ชั่นที่เป็นทางการที่ได้รับอนุญาตโดย Mozilla เท่านั้น
{: #external-parties }

* หากอุปกรณ์ของคุณผลิตโดยผู้ผลิตรายอื่นที่ไม่ใช่ Mozilla ให้ยึดตามนโยบายความเป็นส่วนตัวของผู้ผลิตนั้นๆ
* ระบบปฏิบัติการ Firefox ยินยอมให้คุณใช้บริการจากผู้ให้บริการอื่นๆ เช่น ผู้ให้บริการอีเมลหรือโซเชียลร่วมกันหลายบริการได้ โดยให้ยึดตามนโยบายความเป็นส่วนตัวของบริการเหล่านั้นกับการใช้งานของคุณ
