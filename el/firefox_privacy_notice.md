# Πολιτική απορρήτου περιηγητή Firefox

20 Ιουλίου 2017
{: datetime="2017-07-20" }

Μας ενδιαφέρει η προστασία του απορρήτου σας. Όταν ο Firefox στέλνει πληροφορίες στην Mozilla (δηλαδή εμάς), η [πολιτική απορρήτου μας](https://www.mozilla.org/privacy/) περιγράφει πώς χειριζόμαστε αυτές τις πληροφορίες.

## Τι πρέπει να γνωρίζετε

Ο Firefox συνδέεται αυτόματα με εμάς και τους παροχείς υπηρεσιών μας για να παρέχει ενημερώσεις, ασφάλεια, τμήματα φράσεων, αναφορές υγείας Firefox και άλλες δυνατότητες.
{: #essential-features }

* **Ενημερώσεις περιηγητή και προσθέτων**
{: #auto-updates }

	Ενημερώσεις περιηγητή: Μία φορά την ημέρα, ο Firefox στέλνει τις παρακάτω πληροφορίες στην Mozilla, όπου ελέγχει για ενημερώσεις περιηγητή: τις πληροφορίες έκδοσης του Firefox, την προτίμηση γλώσσας, το λειτουργικό σύστημα και την έκδοσή του. Μπορείτε να [απενεργοποιήσετε τις ενημερώσεις αν ακολουθήσετε αυτές τις οδηγίες](https://support.mozilla.org/kb/how-stop-firefox-automatically-making-connections#w_auto-update-checking) αλλά έτσι μπορεί να μείνετε εκτεθειμένοι σε κενά ασφαλείας.

	Λίστα αποκλεισμού προσθέτων: Ο Firefox επικοινωνεί με την Mozilla μία φορά την ημέρα για να ελέγξει για πληροφορίες προσθέτων ώστε να ελέγξει για κακόβουλα πρόσθετα. Σε αυτές τις πληροφορίες περιλαμβάνονται, για παράδειγμα: η έκδοση του περιηγητή, το Λ.Σ. και η έκδοσή του, η τοποθεσία, ο συνολικός αριθμός αιτημάτων, η ώρα του τελευταίου αιτήματος, η ώρα, η διεύθυνση IP και η λίστα προσθέτων που έχετε εγκατεστημένα. Μπορείτε να [απενεργοποιήσετε τις ενημερώσεις μετα-δεδομένων](https://support.mozilla.org/kb/how-stop-firefox-making-automatic-connections) οποιαδήποτε στιγμή, αλλά έτσι μπορεί να μείνετε εκτεθειμένοι σε κενά ασφαλείας.

* **Τμήματα φράσεων**
{: #snippets }

	Η προεπιλεγμένη αρχική σελίδα του Firefox (&lt;about:home&gt;) φορτώνει μικρά κομμάτια πληροφοριών ακριβώς κάτω από τη γραμμή αναζήτησης που πιστεύουμε ότι θα σας φανούν χρήσιμα. Τα καλούμε "τμήματα φράσεων". Περίπου μία φορά την ημέρα, ο Firefox συνδέεται με την Mozilla και σας παρέχει νέα τμήματα φράσεων, εφόσον υπάρχουν. Η Mozilla μπορεί να συλλέγει πληροφορίες για το πόσο συχνά γίνεται κλικ στα τμήματα φράσεων, το όνομα του τμήματος φράσεως, την τοποθεσία του περιηγητή και την έκδοση του Firefox που χρησιμοποιείτε. Διατηρούμε αυτές τις πληροφορίες μόνο μετά από 60 ημέρες σε συγκεντρωτική μορφή.

	Για να διευκολυνθεί στην εμφάνιση σχετικών τμημάτων φράσεων, ο Firefox στέλνει στην Mozilla ένα αίτημα, μία φορά το μήνα, να αναζητήσει την τοποθεσία σας σε επίπεδο χώρας βάσει της διεύθυνσης IP σας. Στη συνέχεια επιστρέφουμε αυτή την πληροφορία επιπέδου χώρας στον Firefox, όπου αυτή φυλάσσεται τοπικά. Στη συνέχεια, ο Firefox επιλέγει να σας εμφανίζει τμήματα φράσεων βάσει της τοπικά αποθηκευμένης πληροφορίας χώρας.

	Ορισμένα τμήματα φράσεων με χορηγό τη Mozilla είναι διαδραστικά και σας επιτρέπουν να μοιράζεστε, προαιρετικά, τον αριθμό τηλεφώνου ή τη διεύθυνση email σας. Για παράδειγμα, μπορείτε να καταχωρήσετε τον αριθμό τηλεφώνου σας για να λάβετε ένα μήνυμα SMS για να εγκαταστήσετε το Firefox για Android. Τα στοιχεία σας λαμβάνονται και υπόκεινται επεξεργασία από τον προμηθευτή μας μάρκετινγκ για κινητά.

* **Αναφορά υγείας Firefox**
{: #health-report .inproduct-link }

	H αναφορά υγείας Firefox (FHR) έχει σχεδιαστεί για να σας παρέχει πληροφορίες για τη σταθερότητα και την απόδοση του περιηγητή σας, καθώς και συμβουλές υποστήριξης αν αντιμετωπίσετε προβλήματα, όπως πολύ συχνά προβλήματα ή αργούς χρόνους εκκίνησης. Η Mozilla συλλέγει και συγκεντρώνει τα δεδομένα σας μαζί με αυτά από άλλους χρήστες του Firefox και τα επιστρέφει στον περιηγητή σας ώστε να μπορείτε να διαπιστώσετε πώς μεταβάλλεται η απόδοση του Firefox που χρησιμοποιείτε με την πάροδο του χρόνου. Αυτά τα δεδομένα περιλαμβάνουν, για παράδειγμα: συσκευές υλικού, λειτουργικό σύστημα, έκδοση Firefox, πρόσθετα (αριθμός και τύπος), χρόνος εμφάνισης συμβάντων περιηγητή, απόδοση, επαναφορές συνεδριών, διάρκεια συνεδρίας, αλληλεπιδράσεις με σημεία πρόσβασης αναζήτησης και χρήση κωδικών συνεργατών αναζήτησης στο Firefox, παλαιότητα προφίλ, αριθμός προβλημάτων και αριθμός σελίδων. Η FHR δεν στέλνει στη Mozilla τα URL που επισκέπτεστε.

	Χρησιμοποιούμε τα δεδομένα που στέλνονται μέσω της FHR για να παρέχουμε στους χρήστες τις λειτουργίες της FHR, π.χ. για να σας βοηθούμε να αναλύετε και να αντιμετωπίζετε θέματα απόδοσης του περιηγητή σας. Χρησιμοποιούμε επίσης αυτά που μαθαίνουμε από τα δεδομένα της FHR συγκεντρωτικά, για να βελτιώνουμε τον Firefox. Μπορείτε να επιλέξετε να [απενεργοποιήσετε το διαμοιρασμό δεδομένων](https://support.mozilla.org/kb/firefox-health-report-understand-your-browser-perf#w_how-to-turn-data-sharing-on-or-off).

* **Ασφάλεια**
{: #security }

	Ο Firefox ελέγχει αυτόματα για κακόβουλες ή πλαστογραφημένες ιστοσελίδες, χαλασμένα πρόσθετα και πιστοποιητικά SSL έκδοσης τρίτων.

	Ασφαλή πιστοποιητικά ιστοσελίδων: Όταν επισκέπτεστε μια ασφαλή ιστοσελίδα (δηλ. με την ένδειξη "https"), ο Firefox επικυρώνει το πιστοποιητικό της ιστοσελίδας. Σε αυτή την περίπτωση μπορεί να γίνεται και επικοινωνία με κάποιον τρίτο παροχέα που ορίζεται στο πιστοποιητικό. Ο Firefox στέλνει σε αυτό τον τρίτο πληροφορίες με τις οποίες αναγνωρίζεται το [πιστοποιητικό](https://support.mozilla.org/kb/secure-website-certificate) της ιστοσελίδας. Μπορείτε να [αλλάξετε τις προτιμήσεις σας](https://support.mozilla.org/kb/advanced-settings-browsing-network-updates-encryption#w_certificates-tab), αν όμως απενεργοποιήσετε τη δυνατότητα online πιστοποίησης, ο Firefox δεν μπορεί να επιβεβαιώσει την ταυτότητα τις ιστοσελίδας που επισκέπτεστε. Αν απενεργοποιήσετε αυτή τη δυνατότητα μπορεί να αυξηθεί ο κίνδυνος υποκλοπής των προσωπικών σας πληροφοριών. Αν βρεθείτε σε μια [μη έμπιστη σύνδεση](https://support.mozilla.org/kb/connection-untrusted-error-message), μπορείτε επίσης να επιλέξετε να στείλετε στη Mozilla τα σχετικά πιστοποιητικά.

	Προστασία από πλαστογραφία και επιθέσεις στον Firefox: Περίπου δύο φορές την ώρα, ο Firefox κάνει λήψη των λιστών ασφαλούς περιήγησης της Google για καλύτερο αποκλεισμό της πρόσβασης σε ιστοσελίδες και λήψεις που είναι κακόβουλες ή πλαστογραφημένες (η πολιτικής προστασίας απορρήτου της Google βρίσκεται στη διεύθυνση <https://www.google.com/policies/privacy/>). Για όσα εκτελέσιμα αρχεία έχουν ληφθεί αλλά δεν περιλαμβάνονται σε αυτές τις λίστες, ο Firefox μπορεί να στέλνει μετα-δεδομένα, συμπεριλαμβανομένων των URL που σχετίζονται με το αρχείο που λήφθηκε, στην υπηρεσία ασφαλούς περιήγησης. Επισκεφθείτε τη διεύθυνση <https://support.mozilla.org/kb/how-does-phishing-and-malware-protection-work> για να μάθετε περισσότερα για την υπηρεσία ασφαλούς περιήγησης ή να την απενεργοποιήσετε. Αν απενεργοποιήσετε αυτές τις δυνατότητες, ο Firefox δεν μπορεί να σας προειδοποιεί για ενδεχομένως παράνομες ή κακόβουλες ιστοσελίδες ή ληφθέντα αρχεία.

* **Στατιστικά χρήσης** (ονομάζονται και "Τηλεμετρία" σε εσωτερικές εκδόσεις που δεν προορίζονται για κυκλοφορία)
{: #telemetry .inproduct-link}

	Τα στατιστικά χρήσης ή η "Τηλεμετρία" είναι μια δυνατότητα του Firefox που στέλνει στη Mozilla στατιστικά χρήσης, απόδοσης και ανταπόκρισης σχετικά με δυνατότητες της επιφάνειας χρήσης και με τη διαμόρφωση μνήμης και υλικού. Επίσης, συλλέγεται η διεύθυνση IP σας ως τμήμα ενός τυποποιημένου αρχείου καταγραφής web. Τα στατιστικά χρήσης μεταδίδονται μέσω SSL και μας βοηθούν να βελτιώνουμε τις μελλοντικές εκδόσεις του Firefox. Αφού σταλούν στη Mozilla, τα στατιστικά χρήσης συλλέγονται συγκεντρωτικά και γίνονται διαθέσιμα σε ευρεία γκάμα από προγραμματιστές, στους οποίους περιλαμβάνονται τόσο εργαζόμενοι της Mozilla όσο και συνεισφέροντες από το ευρύ κοινό. Όταν είναι ενεργοποιημένη η Τηλεμετρία, ορισμένα βραχυπρόθεσμα πειράματα ενδέχεται να συλλέγουν πληροφορίες για τις ιστοσελίδες που επισκέπτεστε.

	Αυτή η δυνατότητα ενεργοποιείται εξ ορισμού στις εσωτερικές εκδόσεις Nightly και Beta/Developer του Firefox, για να διευκολύνονται αυτοί οι χρήστες να παρέχουν σχόλια στην Mozilla. Στην έκδοση για γενική κυκλοφορία στην αγορά του Firefox, αυτή η δυνατότητα είναι εξ ορισμού απενεργοποιημένη.

	Μπορείτε να [μάθετε περισσότερα για την Τηλεμετρία εδώ](https://support.mozilla.org/kb/send-performance-data-improve-firefox) και για τον τρόπο ενεργοποίησης ή απενεργοποίησής της.

* **Πλακίδια**
{: #searchsuggestions }

	Οι προτάσεις αναζήτησης είναι μια δυνατότητα που σας βοηθά να βρίσκετε συνηθισμένες φράσεις που έχουν ήδη αναζητήσει άλλα άτομα. Αυτές οι προτάσεις αναζήτησης προσφέρονται από τις προεπιλεγμένες μηχανές αναζήτησής σας (όπως οι Google, Yahoo κ.λπ.) και όχι από το Firefox. Αν ενεργοποιήσετε αυτή τη δυνατότητα και η προεπιλεγμένη μηχανή αναζήτησής σας υποστηρίζει τις προτάσεις, το Firefox μπορεί να στείλει τους όρους που πληκτρολογείτε στη γραμμή διευθύνσεων ή στη γραμμή αναζήτησης στην προεπιλεγμένη μηχανή αναζήτησής σας για να ανακτήσει προτάσεις, ενέργεια για την οποία ισχύει η αντίστοιχη πολιτική απορρήτου της προεπιλεγμένης μηχανής αναζήτησής σας. Μπορείτε να [μάθετε περισσότερα για τις προτάσεις αναζήτησης εδώ](https://support.mozilla.org/kb/use-popular-search-suggestions-firefox-search-bar) και για τον τρόπο ενεργοποίησης ή απενεργοποίησής τους.

* **Καταγραφή παραπομπών και καμπάνιας**{: #thirdparty } ** **{: #referraltracking }

	Για να κατανοήσουμε καλύτερα και να βελτιώσουμε τις καμπάνιες μάρκετινγκ που διεξάγουμε, ο Firefox στέλνει ως προεπιλογή ορισμένες πληροφορίες. Σε αυτές τις πληροφορίες περιλαμβάνονται “Δεδομένα Παραπομπών”, όπως το domain της ιστοσελίδας ή η διαφημιστική καμπάνια που σας παρέπεμψε να κάνετε λήψη και εγκατάσταση του Firefox, καθώς “Δεδομένα Αλληλεπίδρασης”, σχετικά με το ποιες δυνατότητες χρησιμοποιείτε στον Firefox.

	__Δεδομένα Παραπομπών__
	Στα Android και iOS, ο Firefox στέλνει Δεδομένα Παραπομπών στον προμηθευτή μας αναλυτικής για κινητά, ενώ περιλαμβάνει επίσης αναγνωριστικό διαφήμισης Google, διεύθυνση IP, χρονοσήμανση, χώρα, τοποθεσία, λειτουργικό σύστημα και έκδοση εφαρμογής. Μπορείτε να μάθετε περισσότερα [εδώ](https://support.mozilla.org/kb/desktop-attribution-privacy), συμπεριλαμβανομένου του τρόπου απενεργοποίησης αυτών των αναφορών.

	Στην επιφάνεια εργασίας, ο Firefox καταγράφει και στέλνει Δεδομένα Παραπομπών στη Mozilla, στο πλαίσιο της αναφοράς υγείας Firefox (FHR). Μπορείτε να μάθετε περισσότερα [εδώ](https://support.mozilla.org/kb/desktop-attribution-privacy), συμπεριλαμβανομένου του τρόπου δήλωσης της μη επιθυμίας σας ή απενεργοποίησης αυτών των αναφορών.

	__Δεδομένα Αλληλεπίδρασης__
	Στο iOS, ο Firefox στέλνει Δεδομένα Αλληλεπίδρασης στην Leanplum, τον προμηθευτή μας μάρκετινγκ μέσω κινητού, η οποία διαθέτει τη δική της [πολιτική απορρήτου](https://www.leanplum.com/privacy/).  Αυτά τα δεδομένα μας επιτρέπουν να δοκιμάζουμε διαφορετικές δυνατότητες και εμπειρίες, καθώς και να παρέχουμε προσαρμοσμένα μηνύματα και προτάσεις για τη βελτίωση της εμπειρίας σας με τον Firefox. Μάθετε περισσότερα για αυτή τη συλλογή δεδομένων [εδώ](https://github.com/mozilla-mobile/firefox-ios/blob/master/MMA.md), καθώς και πώς μπορείτε να [απενεργοποιήσετε τη δυνατότητα αυτή](https://support.mozilla.org/kb/send-anonymous-usage-data-firefox-mobile-devices).

---------------------------------------

Όταν του το ζητήσετε, ο Firefox συνδέεται επίσης στη Mozilla για να σας παρέχει λειτουργίες όπως το Sync, οι υπηρεσίες τοποθεσίας, οι αναφορές προβλημάτων και τα πρόσθετα.
{: #optional-features }

* **Sync**
{: #sync }

	Το [Firefox Sync](https://www.mozilla.org/firefox/sync/) είναι μια υπηρεσία που σας επιτρέπει να κάνετε συγχρονισμό των σελιδοδεικτών, του ιστορικού περιήγησης, των κωδικών πρόσβασης και των ρυθμίσεών σας στον Firefox σε όλες τις συσκευές σας. Αν χρησιμοποιείτε την υπηρεσία Sync, μπορείτε να διαβάσετε την [πολιτική απορρήτου του Firefox Sync](https://accounts.firefox.com/legal/privacy).

* **Υπηρεσίες τοποθεσίας**
{: #location-services }

	Ο Firefox διαθέτει μια λειτουργία που επιτρέπει στις ιστοσελίδες να ζητήσουν την τοποθεσία σας (π.χ. για να μπορούν να σας δείξουν τη θέση σας στο χάρτη). Αν μια ιστοσελίδα ζητήσει την τοποθεσία σας, ο Firefox ζητά την άδειά σας πριν προσδιορίσει και μοιραστεί την τοποθεσία σας. Για να προσδιορίσει την τοποθεσία σας, ο Firefox μπορεί να χρησιμοποιήσει διάφορα δεδομένα, συμπεριλαμβανομένων των δυνατοτήτων γεωεντοπισμού του λειτουργικού σας συστήματος, των δικτύων WiFi, των κυψελών κινητής τηλεφωνίας ή της διεύθυνσης IP σας. Για την εκτίμηση της τοποθεσίας σας χρειάζεται η αποστολή ορισμένων από αυτές τις πληροφορίες στην υπηρεσία γεωεντοπισμού της Google, η οποία διαθέτει τη δική της [πολιτική απορρήτου](https://www.google.com/privacy/lsf.html).

* **Αναφορά προβλημάτων**
{: #crash-reporter .inproduct-link }

	Έχετε τη δυνατότητα να στέλνετε στη Mozilla μια αναφορά προβλημάτων μετά από κάθε πρόβλημα του Firefox. Αυτή η αναφορά περιλαμβάνει τεχνικές πληροφορίες με τις οποίες μπορούμε να βελτιώσουμε τον Firefox, συμπεριλαμβανομένης της αιτίας του προβλήματος του Firefox, του ενεργού URL τη στιγμή που προέκυψε το πρόβλημα, και της κατάστασης της μνήμης του υπολογιστή κατά τη διάρκεια του προβλήματος. Η αναφορά προβλημάτων που λαμβάνουμε μπορεί να περιλαμβάνει προσωπικές πληροφορίες. Κάνουμε τμήματα αναφορών προβλημάτων διαθέσιμα στο ευρύ κοινό στη διεύθυνση <https://crash-stats.mozilla.com/>. Πριν δημοσιεύσουμε αναφορές προβλημάτων σο ευρύ κοινό, λαμβάνουμε συγκεκριμένα μέτρα για να αφαιρέσουμε αυτόματα τις προσωπικές πληροφορίες. Δεν αφαιρούμε οτιδήποτε μπορεί να γράψετε στο πλαίσιο σχολίων.

* **Σφάλματα SSL**
{: #ssl-errors }

	Έχετε τη δυνατότητα να στέλνετε στη Mozilla μια αναφορά σφάλματος όταν διακόπτεται μια ασφαλής σύνδεση με ιστοσελίδα. Αυτή η αναφορά καταγράφει το πιστοποιητικό της ιστοσελίδας καθώς και διαγνωστικούς κωδικούς σφαλμάτων. Αυτές οι πληροφορίες βοηθούν τη Mozilla να παρακολουθεί την αποτελεσματικότητα των "καρφιτσωμένων" πιστοποιητικών ιστοσελίδων και να ανιχνεύει πιθανές επιθέσεις phishing ενάντια στους χρήστες μας.

* **Πρόσθετα**
{: #addons }

	Ο Firefox προσφέρει μια σελίδα Λήψη περισσότερων προσθέτων στη Διαχείριση προσθέτων, όπου προβάλλονται δημοφιλή πρόσθετα και εμφανίζονται εξατομικευμένες προτάσεις βάσει των προσθέτων που έχετε ήδη εγκατεστημένα. Για να εμφανίσει τις εξατομικευμένες προτάσεις, ο Firefox στέλνει πληροφορίες στη Mozilla, συμπεριλαμβανομένης της λίστας των προσθέτων που έχετε εγκατεστημένα, πληροφορίες έκδοσης του Firefox και της διεύθυνσής σας IP. Αυτή η επικοινωνία συμβαίνει μόνο όταν είναι ανοικτή η περιοχή Λήψη περισσότερων προσθέτων και μπορεί να απενεργοποιηθεί αν ακολουθήσετε [αυτές τις οδηγίες](https://support.mozilla.org/kb/how-stop-firefox-making-automatic-connections). Η Διαχείριση προσθέτων στον Firefox έχει ένα πεδίο αναζήτησης όπου μπορείτε να καταχωρήσετε λέξεις-κλειδιά για την εκτέλεση αναζητήσεων και η Mozilla συλλέγει αυτές τις αναζητήσεις με λέξεις-κλειδιά, καθώς και τις πληροφορίες έκδοσης του Firefox, την τοποθεσία και το Λ.Σ. σας για να σας προβάλλει προτάσεις.

* **Ειδοποιήσεις push**
{: #push-notifications }

	Οι ειδοποιήσεις push επιτρέπουν στις ιστοσελίδες να σας στέλνουν ειδοποιήσεις και ενημερώσεις εφόσον βεβαιώσετε ότι επιθυμείτε να τις λαμβάνετε. Για να λαμβάνετε ειδοποιήσεις, το Firefox στέλνει πληροφορίες στην Mozilla σχετικά με τις ιστοσελίδες, από τις οποίες έχετε συμφωνήσει να λαμβάνετε ειδοποιήσεις push. Αποθηκεύουμε αυτές τις πληροφορίες με τέτοια μορφή ώστε να καταστούν ανώνυμες, μαζί με τον αριθμό ειδοποιήσεων που σας στέλνει κάθε ιστοσελίδα. Για να βοηθήσει τους προγραμματιστές να χρησιμοποιούν καλύτερα τις ειδοποιήσεις push, η Mozilla μπορεί να μοιράζεται με ορισμένους προγραμματιστές συγκεντρωτικές πληροφορίες, συμπεριλαμβανομένου του αριθμού επισκεπτών στην ιστοσελίδα τους που έχουν εγγραφεί ή αποχωρήσει ως συνδρομητές για τις ειδοποιήσεις τους push. Μπορείτε να διαχειριστείτε τις ειδοποιήσεις σας push στο Firefox αν ακολουθήσετε [αυτές τις οδηγίες](https://support.mozilla.org/kb/push-notifications-firefox).

	__Αποστολές στιγμιοτύπων οθόνης__
	Όσα στιγμιότυπα οθόνης επιλέγετε να αποστείλετε, στέλνονται στη Mozilla και φυλάσσονται για το περιορισμένο χρονικό διάστημα που επισημαίνεται, το οποίο μπορείτε να αλλάξετε.  Μπορεί να αποκτούμε πρόσβαση στα απεσταλμένα σας στιγμιότυπα οθόνης, όταν αυτό είναι εύλογα αναγκαίο για τη λειτουργία της υπηρεσίας.  Μπορείτε να διαγράψετε τα απεσταλμένα σας στιγμιότυπα οθόνης ανά πάσα στιγμή.  

	__Δεδομένα Αλληλεπίδρασης__
	Λαμβάνουμε δεδομένα όπως οι επισκέψεις στην ιστοσελίδα στιγμιοτύπων οθόνης Firefox, πόσο συχνά γίνεται πρόσβαση στα απεσταλμένα στιγμιότυπα οθόνης και πόσο συχνά γίνεται κοινή χρήση τους από εσάς ή από άλλους, καθώς και οι αλληλεπιδράσεις σας με κουμπιά, πλακίδια και κινήσεις του ποντικιού που σχετίζονται με τη λήψη στιγμιοτύπων οθόνης.  
Για τις επισκέψεις στην ιστοσελίδα στιγμιοτύπων οθόνης Firefox, η [ειδοποίηση απορρήτου ιστοσελίδων](https://www.mozilla.org/privacy/websites/) μας περιγράφει τους τύπους δεδομένων που συλλέγουμε. 

	__Τεχνικά Δεδομένα__
	Λαμβάνουμε δεδομένα όπως το μέσο μέγεθος και ο αριθμός των στιγμιοτύπων οθόνης που στέλνετε, η έκδοση του περιηγητή Firefox που χρησιμοποιείτε, το λειτουργικό σύστημα της συσκευής και τα σφάλματα.  Η διεύθυνση IP από την οποία γίνεται πρόσβαση στην ιστοσελίδα στιγμιοτύπων οθόνης Firefox συλλέγεται προσωρινά, ως τμήμα ενός τυποποιημένου αρχείου καταγραφής web. 

	Διαβάστε την πλήρη τεκμηρίωση [εδώ](https://github.com/mozilla-services/screenshots/blob/master/docs/METRICS.md) ή διαγράψτε όλα τα στιγμιότυπα οθόνης σας [εδώ](https://screenshots.firefox.com/leave-screenshots).

Με εξαίρεση τις περιπτώσεις στις οποίες σημειώνεται κάτι διαφορετικό, η παρούσα πολιτική απορρήτου αφορά τις πιο πρόσφατες εκδόσεις για γενική κυκλοφορία του Firefox. Οι προεκδόσεις μας (Beta, έκδοση Developer, Nightly και TestFlight) βρίσκονται ακόμα σε φάση ενεργής ανάπτυξης και μπορεί να περιέχουν νέες δυνατότητες ή να έχουν διαφορετικά χαρακτηριστικά απορρήτου. Οι προεκδόσεις στέλνουν αυτόματα [Δεδομένα τηλεμετρίας](https://gecko.readthedocs.io/en/latest/toolkit/components/telemetry/telemetry/index.html) στην Mozilla, τα οποία μας βοηθούν να βελτιώσουμε το Firefox.
